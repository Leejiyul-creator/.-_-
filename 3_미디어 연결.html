미디어 연결

1. 동영상 연결

- video 태그 사용

  • src: 연결
  • width: 넓이
  • height: 높이 (미입력시 가로와 정비례로 자동 설정된다.)
  • controls: 조정판넬
  • autoplay : 자동재생
  • muted: 무음처리 (자동재생에 필수조건)
  • loop: 반복
  • playinline: 팝업실행차단

- iframe 태그 사용

  • iframe 태그는 다른 html 문서를 연결하여 보여주는 태그
  • 소스코드를 복사하여 붙여넣기

2. 오디오 연결

- 오디오 태그 사용

  • src: 연결
  • controls: 조정판넬
  • autoplay : 자동재생 - chrome에서 제한
  • muted: 무음
  • loop: 반복

3. 지도 연결

- ifram 태그 사용 : google에서 주로 사용한다.

  • 구글 지도 검색
  • 공유 버튼 클릭
  • 지도 퍼가키 탭을 클릭하여 HTML 복사

- 스크립트 사용 : daum(kakao)에서 주로 사용 (권장)

  • 다음지도 검색
  • 위치 표시 상단에 있는 펼치기 버튼 클릭
  • 공유버튼 클릭 후 HTML 태그 복사
  • 지도 크기 지정한 후 소스 생성하기 클릭
  • 태그를 복사하여 붙여넣기

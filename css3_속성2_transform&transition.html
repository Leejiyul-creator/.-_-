transform&transition(P.448)

transform은 변형 / transition은 과정

  • translate

    • 왼쪽 위를 기준으로 이동
    • tranosform; translate(x값, y값)

  • rotate

    • 시계방향으로 각도 회전
    • deg 단위 사용
    • -(마이너스)값 사용 가능
    • transform: rotate(deg)형식으로 입력

  • scale

    • 기준은 1(=100%)
    • transform scale(배수);
      transform scale(x값, y값)형식으로 입력
    • transform scale()
      transform: scale()transform scale() 둘로 나누어 사용 가능

  • skew

    • transform: skew(x값, y값) 형식으로 입력
    • 값을 하나만 입력했을 때는 x값으로 인식
    • tranosform: skew
      transform: skew() 둘로 나누어 사용 가능
    • deg 단위 사용

ico 파일 제작

  1. https://convertico.com/favicon에 방문하여 png파일을 ico파일로 변환하여 저장
  2. head 태그 안에
    <link rel="shortcut icon" href="파일이름.ico" tyoe="image/x-icon">
    태그를 삽입
  3. 고해상의 ico파일을 원할 대에는 사이트 내의 https://convertico.com턉을 클릭하여 ico 파일을 제작

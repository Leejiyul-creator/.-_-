폰트 사용하기

English Font와 한글서체를 구분해서 사용할 때에는 영문서체를 먼저 지정 - 한글서체는 영문을 지원하지만 English Font는 한글 지원 불가

1. 기본PC(내장) 폰트 사용 - 기본적인 서체 권장

font-family 속성을 사용하여 우선순위별로 2가지 이상 지정

2. 폰트 업로드 - 반드시 저작권 확인 후 사용

@font-face 사용방법

  1. 글꼴을 다운로드하거나 구매한 뒤 폴더를 지정하여 업로드
  2. css에 @font-face {font-family: '글꼴이름', src: url(폰트파일저장위치) format('포맷');) 설정
  3. 선택자에 font-family 설정

3. webfont 사용 (권장)

webfont 사용 방법

  1. http://fonts.google.com사이트에서 서체를 선택
  2. 오른쪽 슬라이드 메뉴 창에서 Embed탭 선택
  3. Style 항목에서 + Select this style 클릭
  4. 하위의 <link>(권장) 혹은 @important 탭을 사용하여 적용

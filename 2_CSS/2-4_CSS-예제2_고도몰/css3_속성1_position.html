CSS 속성 - position(P.341)

position 속성

  • static

    기본 설정 - 고정위치

    • 기본 설정이므로 별도로 설정하지 않는다.
    • 위치값의 기준은 브라우저의 body 영역
    • left : bottom: 등 특별 위치값도 사용 가능

  • relative

    상대적 위치

    • left : top: 등 위치값 사용
    • 위치값의 기준은 현재 위치
    • z값의 우선순위 높음

    • absolute

    • 상위요소 중 relative 속성을 가진 요소가 있으면 그 요소 위치를 기준으로 삼음

  • absolute

    절대적 위치

  • fixed

    절대적 고정 위치

html+css 입력방법

선택자 {속성: 속성값;}


css 입력 방법


선택자 유형

  1. type(태그) - '태그명'을 그대로 입력
  2. class - '.' + '클래스명'을 입력
  3. id - '#' + '아이디명'을 입력
  4. 복합

선택자는 주체적일수록 우선순위가 높다.


CSS를 HTML에 적용시키는 방법

1. Inline Style Sheet

html 문서 안에 style 속성을 직접 입력하는 방식
- css 내용이 매우 적을 때 사용
- 우선 순위 높음

2. Internal Style Sheet

HTML 문서 HEAD 태그 안에 style 태그를 삽입
- 간단한 문서에 사용
- style 태그에 type 속성을 입력

3. Linking Style Sheet

별도의 css파일을 작성하여 HTML문서와 연결
- head 태그 안에 link 태그를 입력한 후 href, rel, type 속성을 지정
(가장 일반적인 방법)
